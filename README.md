# Requisitos.
- Version de **openjdk** > 17.
- Maven.

# Empezar.

## Clonar repositorio.

- Usando git.
```sh
git clone "https://github.com/Lauta-dev/tp-java.git"
```

- Usando gh cli
```sh
gh repo clone "Lauta-dev/tp-java".
```
## Ejecutar programa
```sh
mvn package; java -cp ./target/tp-1.0-SNAPSHOT.jar tp.Main
```
